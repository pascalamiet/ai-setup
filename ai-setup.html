üß∞ Prerequisites

Before you summon the AI overlords into your terminal, make sure you have the basics. Think of this as the "have you tried turning it off and on again?" of the AI world.

üçé macOS

1 ‚Äî Install Homebrew (if you don't have it)

Homebrew is macOS's package manager. If you already have it, skip this step. Run brew --version to check.
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

2 ‚Äî Install Node.js via Homebrew

Node.js is required if you want to install tools via npm. Homebrew-installed CLI tools bundle their own runtime, so this is optional if you go the Homebrew route for everything.

# Verify ‚Äî should print v22.x.x or higher
node --version
ü™ü Windows

1 ‚Äî Install Node.js

Download the LTS installer from nodejs.org, or use winget:

# Verify in a new terminal window
node --version
npm --version

2 ‚Äî Fix PowerShell execution policy (if npm commands fail)

‚ö†Ô∏è Windows sometimes refuses to run npm scripts. Fix it once and forget it:
Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser

OpenAI Codex CLI

Your AI pair programmer who never asks for a raise, but does ask for tokens.

üçé macOS

Option A ‚Äî Homebrew Recommended

Bundles its own Node.js runtime. No manual dependency juggling required.
brew install codex

Option B ‚Äî npm (requires Node.js 22+)

npm install -g @openai/codex
ü™ü Windows

Option A ‚Äî WSL 2 Recommended

Native Windows support is experimental. WSL 2 is the stable path. Run these in an elevated PowerShell:
# Step 1 ‚Äî Install WSL 2
wsl --install
# Step 2 ‚Äî Open WSL, then install nvm + Node.js 22
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/master/install.sh | bash
# Step 3 ‚Äî Restart WSL shell, then:
nvm install 22
# Step 4 ‚Äî Install Codex
npm install -g @openai/codex

Option B ‚Äî Native Windows Experimental

‚ö†Ô∏è Requires Visual Studio Build Tools (C++ workload). Approach with care and a coffee.
winget install --id Microsoft.VisualStudio.2022.BuildTools -e
npm install -g @openai/codex

üîë Authentication

You need a ChatGPT Plus/Pro/Team/Enterprise account, or an OpenAI API key.
# Launch and follow the interactive prompt
codex
# Or set your API key directly (add to ~/.zshrc or ~/.bashrc)
export OPENAI_API_KEY="sk-your-key-here"

Anthropic Claude Code

The tool you're currently interacting with. It's watching. It's always watching.

üçé macOS

Option A ‚Äî Native installer Recommended

Auto-updates itself. No Node.js needed. The cleanest route.

Option B ‚Äî Homebrew

‚ö†Ô∏è Does not auto-update. You'll need to run brew upgrade claude-code manually.

# Upgrade later with:

Option C ‚Äî npm Deprecated

‚ö†Ô∏è The npm method is deprecated. Only use if you have a strong opinion and enjoy pain.
# Requires Node.js 18+. Do NOT use sudo here.

ü™ü Windows

Option A ‚Äî PowerShell one-liner Recommended

Auto-updates. No Node.js needed. The cleanest Windows install.

Option B ‚Äî winget

# Upgrade later with:
winget upgrade Anthropic.ClaudeCode

Option C ‚Äî WSL

# Inside your WSL terminal:

‚ö†Ô∏è Requires a Claude Pro, Max, Teams, or Enterprise plan. The free tier waves at you from a distance.
